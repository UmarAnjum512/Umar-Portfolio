Customer

User Panel

User system ka end customer hota hai. User apna account register aur login karta hai. Apni personal profile details maintain karta hai. Available internet aur telephone plans dekh sakta hai. Online ya retail outlet se service order place karta hai. Apne placed orders ki current status check karta hai. Connection feasibility aur approval status dekh sakta hai. Bill details aur invoice view kar sakta hai. Payment history aur paid amount track karta hai. Outstanding dues aur due dates check karta hai. Connection active, inactive ya suspended status dekh sakta hai. Service usage aur account information monitor karta hai.

Manager

Admin Panel

Admin system ka overall control handle karta hai. Employees ke accounts create, update aur delete karta hai. Retail outlets aur service branches manage karta hai. Vendors aur third-party suppliers ka record maintain karta hai. Internet aur telephone plans define karta hai. Plans ke prices, duration aur features set karta hai. Equipment aur products ka inventory manage karta hai. Stock availability aur usage track karta hai. System users ke roles aur permissions control karta hai. Reports aur system data monitor karta hai. Pure system par full administrative authority hoti hai

Retail Staff

Employee Panel

Employee customers ke sath direct interaction karta hai. New customers ka data system me enter karta hai. Customer ke liye new service orders place karta hai. Available plans customer ko explain karta hai. Orders ki current status customer ko batata hai. Bills ko view karta hai lekin modify nahi kar sakta. Customer se payments receive karta hai. Payments system me enter karta hai for record. Equipment handover ka record maintain karta hai. Customer complaints aur requests log karta hai. Plans, vendors ya employees edit nahi kar sakta. Limited access ke sath day-to-day operations handle karta hai.

Technical Staff

Technical Panel

Technical staff connection feasibility check karta hai. Area survey karke service availability confirm karta hai. Order ko technically approve ya reject karta hai. Approved orders par connection activate karta hai. Temporary disconnection requests handle karta hai. Permanent disconnection cases manage karta hai. Installed equipment ki details update karta hai. Connection status ko system me update karta hai. Technical remarks aur logs maintain karta hai. Field work aur backend data synchronize karta hai. Technical issues aur faults record karta hai. Network service quality ensure karta hai.

Accounts Department

Accounts Panel

Accounts department billing aur finance handle karta hai. Approved orders ke liye bills generate karta hai. Customer ke selected plan ke charges calculate karta hai. Applicable tax (12.24%) bill me add karta hai. Discounts aur schemes apply karta hai. Final payable amount calculate karta hai. Bills ko update karne ka authorized access hota hai. Payments ko verify aur approve karta hai. Payment status ke mutabiq bill update karta hai. Invoice aur payment history maintain karta hai. Financial reports aur summaries generate karta hai. System ki financial accuracy ensure karta hai

🛠 Tech Stack Used:

  • Frontend:
    HTML5, CSS3, Bootstrap (Grid & Layouts), JavaScript (jQuery), Chart.js for data visualization.

  • Backend:
    ASP.NET Core MVC (C#).

  • Database:
    SQL Server (Entity Framework Core).

  • Session Management:
    Secure multi-user authentication system (Customers, Admins) using ASP.NET Sessions.
